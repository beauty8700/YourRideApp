# YourRide Frontend

This is the React + TypeScript + Vite frontend for the YourRide application.

## Setup
1. `npm install`
2. `npm run dev`

## Features
- Modern, clean UI inspired by Uber/Ola/Rapido
- OpenStreetMap integration via Leaflet
- Real-time ride status via Sockets
- User and Driver portals
- Responsive design
